RSA Keon Agent SDK v 5.1 unsupported platforms
--------------------------------------------------------------------------

This is a distribution of unsupported builds of the RSA Keon Agent SDK v 5.1 
(referred to as the SDK). "Unsupported" means that these builds have not received 
full QA and that RSA Security does not support them.

This archive contains:

sdk_51r20p1-sol26-intel-dom.tar.gz
SDK on Sun Solaris 2.6 for intel x86 or compatibles

sdk_51r21p1-sol27-intel-dom.tar.gz
SDK on Sun Solaris 2.7 for intel x86 or compatibles

sdk_51r23p1-sol27-sparc-dom.tar.gz
DK on Sun Solaris 2.7 for Sun SPARC

sdk_51r24p1-hpux11-pa-dom.tar.gz
SDK on HP-UX 11.0 for HP-PA machines

sdk_51r25p1-aix432-rs6000-dom.tar.gz
SDK on IBM AIX 4.3.2 on RS6000

sdk_51r26p1-fbsd34-intel-dom.tar.gz
SDK on FreeBSD 3.4 on intel x86 or compatibles

sdk_51r27p1-rh61-intel-exp.tar.gz
SDK on Red Hat Linux 6.1 on intel x86 or compatibles

sdk_51r28p1-nt5-intel-dom.zip
SDK on Windows 2000 on intel x86 or compatibles
