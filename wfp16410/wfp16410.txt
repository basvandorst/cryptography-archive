Description
===========
WinFront for PGP 4 puts a friendly and powerful front-end
over DOS/Win32s PGP. It adds sophisticated key management,
internet key server access, easy configuration, encryption
packages, and output to e-mail. Please see the home page
for complete details.

It is available in native 32 bit and 16 bit versions. 
WinFront for PGP will run on machines from Windows 3.1 up
to Windows NT.

WinFront for PGP is shareware and costs $30.00.


Multiple Language Support
=========================

This version contains support for interchangeable
language modules. Languages available include
English, German, and Dutch. More are being dveloped.

System Requirements
===================
 3MB free disk space
 Internet Connection suggested
 One of
    -Windows 3.1x or higher (for 16bit version)
    -Windows 95 or higher (for 32bit version)
    -Windows NT Service Pack 1 or higher (for 32bit version)
 PGP must already be installed on your computer

Installation Instructions
=========================
Run the file setup.exe to install WinFront for PGP.
This will install the product and create appropriate
Program Manager / Start Menu groups.
