		       The Data Encryption Algorithm
			      Release v.2.01


		       Release Date: December 1st, 1993

		 _________Notification of Changes___________
			  -----------------------
			       From v.2.00


This document describes the changes which have taken place since the
release of the DEA v.2.00 in November 1993.  Please note that versions
2.00 and 2.01 ARE COMPLETELY COMPATIBLE WITH ONE ANOTHER.


The following have been added / altered / modified:

DEA.EXE

	       - CMD line mode tells user into which directory DEA.LOG file
		 is written (default directory)

	       - DOS Error Return Codes now implemented

	       - modified the modulus operator to a longer form which is
		 10 - 12% faster than in v.2.00

	       - miscellaneous minor cosmetic changes


MFD.EXE
	       - will now also delete READ_ONLY files


DEA.DOC
	       - small additions / edited paragraphs / other information





DEA2TECH.DOC     - scheduled for release on, or before December 14, 1993 -

	       - this is the DEA Technical Summary Paper which outlines
		 the DEA components and operations, some 'C' language
		 source code will also be presented for the Cipher Function,
		 and the mod-mult-random-feed algorithm; except for the
		 MMRF, the code presented is non-runable
	       * to be released on, or before December 14, 1993
	       * will be available on the INTERNET sci.crypt conference
	       - also available as a separate file identified as:
		 DEA2TECH.ZIP
