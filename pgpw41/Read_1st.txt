**************************************
	Installing WinPGP® 4.1
**************************************

1. Copy the file, WINPGP4.EXE to a
temporary directory.

2. Double click on the executable from
File Manager.

3. WinPGP® 4.1 is a self installing 
executable. 

4. Read the Setting Up section in the
WinPGP® 4.1 Help File.
