Testing ACE/Agent Authentication

To help you troubleshooting authentication problems, the ACE/Agent software installation includes a small program called acetest. This program tests your ACE/Agent installation to make sure that the ACE/Agent is operating is communicating properly with the ACE/Server.

To test your ACE/Agent environment:

  1. cd to your /etc directory.
        
  2. Create a file called sdace.txt and type the following line in the file:
        
    VAR_ACE= <your VAR_ACE directory>

    your VAR_ACE directory must be the directory that you specified as the VAR_ACE directory during ACE/Agent installation.
             
  3. cd to the netscape_home_directory/plugins/aceagent directory.
       
  4. Run the acetest program.
