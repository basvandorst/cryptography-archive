Uninstalling the ACE/Agent Software

To uninstall ACE/Agent for Netscape Servers:

  1. cd to the netscape_home_directory/plugins directory.

    2.  
  2. Remove the aceagent directory.

    3.  
  3. cd to the netscape_home_directory/docs directory.

    4.  
  4. Remove the aceagent directory.

    5.  
  5. cd to the netscape_home_directory/http[s or d]-hostname/config directory.

    6.     
       
    • If you have made no modifications to the server's obj.conf file since you installed the ACE/Agent software, delete the existing obj.conf file and rename the obj.save file to obj.conf.

      •  
    • If you have made modifications to the obj.conf file, you must edit the file to remove the following lines:

      • Init fn=load-modules shlib=netscape_home_directory/plugins/aceagent/aceagent.so funcs="securidinit,securidnametrans,securidauthform,securidauthorize,securidadmin"
       
          . . .
       
      Init fn=securidinit fname="netscape_home_directory/plugins/aceagent/aceagent.cfg"

          . . .
       
      <Object name=default>
      NameTrans fn=securidnametrans
      Service fn="securidauthorize" type="magnus-internal/securidauth"
      </Object>
       
          . . .
       
      <Object name=cgi>
      Service fn="securidauthorize" type="magnus-internal/securidauth"
      </Object>


  6. Stop and restart the Web server.
