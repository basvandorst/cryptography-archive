ACE/Server Security System - Overview

The Security Dynamics ACE/Server software provides identification and authentication of users on TCP/IP networks.

The ACE/Server product works with SecurID tokens to authenticate the identity of users, granting access only to authorized users on the registered ACE/Agent. When a user enters a SecurID PASSCODE in the WebID authentication form, the ACE/Agent sends an authentication request the ACE/Server, where the PASSCODE is validated.

ACE/Agent for Netscape Servers v1.1 is supported under ACE/Server v1.3 and later.

Note: You must have at least a trial version of the ACE/Server software running on your network in order to utilize ACE/Agent for Netscape Servers v1.1.

For more information about obtaining ACE/Server software or SecurID tokens, visit the Security Dynamics Web site.
