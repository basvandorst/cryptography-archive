Requirements for WebID Users

SecurID Tokens

Users who will authenticate through the WebID authentication forms must have SecurID tokens. Read Preliminary ACE/Server administration tasks for more information about registering users and their tokens in the ACE/Server database.

Browsers

Security Dynamics supports Netscape Navigator versions 1.1 or later and Microsoft Internet Explorer versions 3.0 or later browsers for authentication through the WebID authentication forms. Other browsers may work, but you should consult their manufacturers to make sure the browsers support HTML forms and Persistent Client State HTTP Cookies.

Authentication Instructions

New users will need copies of the WebID authentication instructions, which are in the /netscape_home_directory/docs/aceagent/authdocs directory. These files are in Adobe Acrobat format and are called pinpad.pdf (for users with PINPAD tokens), softid.pdf(for users with SoftIDs), and standard.pdf (for users with Standard tokens or Key Fobs).

You can also provide users with online authentication help by creating hyperlinks to the HTML help files that are included with ACE/Agent for Netscape Servers v1.1.
