ACE/Agent for Netscape Servers Documentation

The following documentation files were copied into the /netscape_home_directory/docs/aceagent directory of your Netscape Web server machine during installation:

  • index.html - the starting page of the ACE/Agent for Netscape Servers v1.1 Administration interface, which contains the ACE/Agent online documentation and the ACE/Agent Administration applet. You must use Netscape Navigator v3.0 to run the Administration applet.
     
  • /authdocs/pinpad.pdf - Authentication instructions in Adobe Acrobat format for users who have SecurID PINPAD tokens. Download the free Acrobat reader here
     
  • /authdocs/softid.pdf - Authentication instructions in Adobe Acrobat format for users who have Security Dynamics SoftIDs. Download the free Acrobat reader here.
       
  • /authdocs/standard.pdf - Authentication instructions in Adobe Acrobat format for users who have SecurID Standard Card or Key Fob tokens. Download the free Acrobat reader here.
     
  • /authdocs/tokenhelp/* - Authentication instructions for all three token types in HTML format. You can provide these instructions to users by creating hyperlinks in the WebID authentication form templates.
