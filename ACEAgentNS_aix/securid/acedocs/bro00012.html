Browser Message:

Invalid Browser State. Either your browser does not support "HTTP Cookies" or you took too long to respond to the [New PIN or Next PRN] request.

Meaning: Either the user waited too long to submit the form or the user does not have a browser that supports HTTP cookies.

Solution: Users must respond to the New PIN or Next PRN forms within 5 minutes or the forms wll time out. In addition, the user should be using a Web browser that supports HTTP cookies (Netscape Navigator v 2.0 or later and MS Internet Explorer v 3.0 or later).
