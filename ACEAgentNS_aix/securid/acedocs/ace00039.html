Configuring Global ACE/Agent Protection Settings

The ACE/Agent Administration applet also lets you configure the various global settings that will affect each protected directory and file. These settings include:

  • cookie distribution
     
  • cookie expiration control
     
  • domain cookie distribution
     
  • connection security options
     
  • precautions that guard against replay attacks

Note: You must stop and restart your Web server in order for any configuration changes to take effect.
