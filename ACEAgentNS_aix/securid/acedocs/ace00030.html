virtual World Wide Web server

A Web server that has been assigned multiple IP addresses. A separate home (root) directory is also assigned to the virtual Web server.

If you are using ACE/Agent for Netscape Servers v1.1 to protect files on a virtual Web server, you must define a secondary ACE/Client node for the virtual server’s IP address. This is done using the ACE/Server sdadmin program. This program is covered in detail in the ACE/Server Administration Manual.
