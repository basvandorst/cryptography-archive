Protecting the Entire Server

To enable Web protection on an entire server:

  1. Start the ACE/Agent Administration applet.
     
  2. Click Select files to secure. The Select Files to Secure dialog box opens. The Select Files to Secure dialog box is laid out as a standard directory tree, with the / icon at the top of the tree indicating the root or “home” directory of the server. You can browse the contents of the server by single-clicking the + and - boxes next to the folder icons.
        
  3. Select the / directory (the root directory of the tree).
     
  4. Click the Secure button. A SecurID icon appears at the top of the file tree, indicating that every page on the server is SecurID protected.
     
  5. Click OK to confirm the changes.

Note: Any files or directories that you add to the server in the future will automatically be SecurID protected.
