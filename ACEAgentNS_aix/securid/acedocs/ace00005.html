System Requirements

In order for ACE/Agent protection to be available on your Web server, the following requirements must be met:

  • ACE/Server v 1.3 or later must be running on your network.
     
  • The Web server machine should have at least 64 megabytes of physical memory.
     
  • Security Dynamics strongly recommends that the Web server machine have a Secure Sockets Layer (SSL) site certificate installed to ensure that transmission of critical information is not compromised.
    Note: For more information about how to obtain an SSL site certificate from a Certificate Authority, visit the VeriSign Corporation's Website.
     
  • To maximize performance, the machine running the ACE/Agent for Netscape Servers software should not be a master or slave ACE/Server.
     
  • Only one version of the ACE/Agent for Netscape Servers software can exist on a single machine. Any future upgrades will overwrite existing versions of the software.
     
  • To administer the ACE/Agent settings, you must have a Java-enabled Web browser such as Netscape Navigator v3.0 or Microsoft Internet Explorer v3.01. ACE/Agent protection can be administered either locally or from a remote machine.
    Note:     Security Dynamics recommends that remote administration of the ACE/Agent settings be done only over a secure connection (https).
     
  • Security Dynamics also recommends that your Web server machine be located in a secure place where unauthorized persons cannot access it.
