Receiving a System-Generated PIN with a Standard Card or Key Fob

If your administrator has instructed you to receive a system-generated PIN:

  1. Initiate a login session. The system will prompt you to enter a PASSCODE.
    • If you have never received a PIN before, in Enter PASSCODE field, type the code that is currently displaying in your SecurID token.
    • If your token previously had a PIN and the administrator did not clear it when setting it into New PIN mode, enter the old PIN followed by the code that is currently displaying in your SecurID token.
  2. Click Send.
    If you have entered the code incorrectly, the system will display the SecurID authentication prompt. Wait for the code in the LCD of the token to change, then try again to log on.
    Once you have entered a valid tokencode, the New PIN dialog box will display.
  3. Make sure that no one can see your screen, and click the System Generated PIN button.
    A system-generated PIN is displayed for 10 seconds.
  4. Wait for the next tokencode to display on you token, and then follow the instructions in SecurID Authentication.
