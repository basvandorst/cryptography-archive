SecurID Tokens

Most SecurID tokens are small, handheld devices containing microprocessors that calculate and display unpredictable codes. These codes change at a specified interval, typically every 60 seconds.

To gain access to protected URLs, a user must enter a valid SecurID PASSCODE, which is made up of two factors:

  • the user’s secret, memorized personal identification number (PIN)
     
  • and the current tokencode generated by the SecurID token

Token types

There are currently five types of SecurID tokens:

  • the standard SecurID Card
  • the Key Fob
  • the PINPAD
  • the SoftID (a software-based token)
  • the SecurID modem (a V34 modem)

See Requirements for WebID Users for more information about distributing token instructions to users.
