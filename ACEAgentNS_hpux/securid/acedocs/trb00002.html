Startup Message:

error: Did not include fname=file in Init fn=securidinit

Meaning: The Init fn=securidinit directive in the obj.conf file did not specify an aceagent.cfg file.

Solution: Edit the obj.conf file and add the line: fname=/netscape_home_directory/plugins/aceagent/aceagent.cfg, where netscape_home_directory is the path to the Web server directory.
