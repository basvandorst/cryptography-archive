Startup Message:

error: ACE/Agent is configured to require SSL encryption, but the Web server is not

Meaning: The web server is not running https with a Secure Sockets Layer enabled.

Solution: Run the server with SSL enabled or disable the Require secure connection to access protected pages option in the ACE/Agent Administration applet (or edit the aceagent.cfg file and set the RequireSecure parameter equal to 0).

If you need more information about how to obtain an SSL certificate, visit a certificate authority such as VeriSign Corporation.
