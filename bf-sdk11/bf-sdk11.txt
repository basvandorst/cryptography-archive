BF-SDK Version 1.1
~~~~~~~~~~~~~~~~~~
software development kit for 16bit DOS and
Windows applications:
* supports the Blowfish encryption algorithm
  designed by Bruce Schneier
* ECB and CBC implementations
* FAST! Up to 1.8 MB/sec on a Pentium P90
* variable number of encryption rounds  
* easy-to-use interfaces for MASM/TASM, Turbo/
  Borland Pascal and MS Visual C++ 1.x
* full source included
