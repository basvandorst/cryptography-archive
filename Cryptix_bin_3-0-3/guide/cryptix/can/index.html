Cryptix TM Application Notes
Cryptix Application Notes (CANs) are discussions of some aspect of usage of Cryptix that doesn't fit well into the javadoc inline documentation. It is optimal to deliver all documentation as close to the code as possible, but some subjects cry out for special treatment. Those special CANs so far are:
  • Comparing performance of 5 hash functions

  • Comparing implementation of 6 symmetric block ciphers

  • Implementing LOKI'91 in Java

  • A simple test data interpreter
